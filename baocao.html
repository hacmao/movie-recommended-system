Collaborative Filter¶

Parameters of ALS Model in PySpark realization are following:¶

  1. NumBlocks is the number of blocks the users and items will be partitioned into in order to parallelize computation.
  2. rank is the number of latent factors in the model.
  3. maxIter is the maximum number of iterations to run.
  4. regParam specifies the regularization parameter in ALS.
  5. implicitPrefs specifies whether to use the explicit feedback ALS variant or one adapted for implicit feedback data (defaults to false which means using explicit feedback).
  6. alpha is a parameter applicable to the implicit feedback variant of ALS that governs the baseline confidence in preference observations (defaults to 1.0).

from pyspark.sql.functions import col, column
from ast import literal_eval

def json_load(s) :
    try :
        return literal_eval(s.asDict()['value'])
    except Exception as e :
        print(e)
        return None
    
def json_load2(s) :
    try :
        return literal_eval(s)
    except Exception as e :
        print(e)
        return None
    
def getAmount() : 
    return sparkSession.sql("select count(*) from ds")\
            .rdd.map(lambda a : a['count(1)']).collect()[0]

def getNewRecords() : 
    global oldCount
    data = sparkSession.sql("SELECT * from ds")
    res = data.rdd.zipWithIndex().filter(lambda c : c[1] >= oldCount) \
            .map(lambda col: json_load(col[0])) \
            .filter(lambda col : col != None) \
            .collect()
    oldCount = getAmount()
    return res
