Kiến trúc tổng quan¶

Hệ thống được thiết kế theo mô hình lambda, chia ra thành các phần chính :

  1. Kafka : truyền dữ liệu của người dùng bao dưới dạng json bao gồm các trường userId, movieId, rating
  2. Spark : nhận dữ liệu từ kafka, tạo model ML và đưa kết quả cho người dùng
  3. model ML
    3.1. Simple model : giựa vào dữ liệu streaming đưa ra các bộ phim tương tự
    3.2. Collaborative filtering : dựa vào dữ liệu rating nghiên cứu xu hướng người dùng
  4. HDFS : lưu dữ liệu phân tán
  5. Kubernetes : toàn bộ các thành phần trên được triển khai trên hệ thống kubernetes

Collaborative Filter¶

Parameters of ALS Model in PySpark realization are following:¶

  1. NumBlocks is the number of blocks the users and items will be partitioned into in order to parallelize computation.
  2. rank is the number of latent factors in the model.
  3. maxIter is the maximum number of iterations to run.
  4. regParam specifies the regularization parameter in ALS.
  5. implicitPrefs specifies whether to use the explicit feedback ALS variant or one adapted for implicit feedback data (defaults to false which means using explicit feedback).
  6. alpha is a parameter applicable to the implicit feedback variant of ALS that governs the baseline confidence in preference observations (defaults to 1.0).

Kafka¶

Một số hàm cơ bản để xử lý với luồng stream :

  1. json_load : phân tích dữ liệu tới dưới dạng json và chuyển về list
  2. getAmount : trả về số thông tin hiện có trong memory
  3. getNewRecords : trả về những record mới toiws
In [24]:
from pyspark.sql.functions import col, column
from ast import literal_eval

def json_load(s) :
    try :
        return literal_eval(s.asDict()['value'])
    except Exception as e :
        print(e)
        return None
    
def json_load2(s) :
    try :
        return literal_eval(s)
    except Exception as e :
        print(e)
        return None
    
def getAmount() : 
    return sparkSession.sql("select count(*) from ds")\
            .rdd.map(lambda a : a['count(1)']).collect()[0]

def getNewRecords() : 
    global oldCount
    data = sparkSession.sql("SELECT * from ds")
    res = data.rdd.zipWithIndex().filter(lambda c : c[1] >= oldCount) \
            .map(lambda col: json_load(col[0])) \
            .filter(lambda col : col != None) \
            .collect()
    oldCount = getAmount()
    return res

Trong trường hợp này, ta mô phỏng dữ liệu người dùng được truyền từ kafka tới với topic là messages. Thực hiện dùng pyspark để lắng nghe luồng stream này. Với mỗi 5s, chúng ta sẽ tiến hành ghi ra memory với view là ds.

# create new stream subcribe topic "messsages" to receive lastest messages
df = sparkSession.readStream.format("kafka") \
    .option("kafka.bootstrap.servers", "kafka:9092")\
    .option("subscribe", "messages")\
    .option("startingOffsets", """{"messages" : {"0" : -1}}""") \
    .load()

# write to memory each 5s 
ds = df.selectExpr("CAST(value AS STRING)")
ds.writeStream \
    .queryName("ds").format('memory') \
    .trigger(processingTime='5 seconds') \
    .start()
